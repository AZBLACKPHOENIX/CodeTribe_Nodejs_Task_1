Mock Media Server API

This server allows you to create, read, update, and delete user data stored in a JSON file.

Endpoints

POST /

Adds a new user.

JSON Format: 
{
  "firstname": "Phoenix",
  "surname": "Sage",
  "age": 25,
  "id": "123"
}

PUT /id

Updates an existing user.

JSON Format: 
{
  "id": "123",
  "firstname": "Johnny",
  "age": 26
}

DELETE /id

Deletes a user by ID.

JSON Format: 
{
  "id": "123"
}

GET /

Returns this documentation page.

Notes

  • All data is stored in user_db.json
  • Make sure to provide all required fields when adding or updating users
  • Use tools like Postman or curl to interact with the API
